https://localhost:7268 - фронт для просмотра уведомлений о действиях с конфигом. Для просмотра уведолений необходимо авторизоваться, для этого заведены 2 пользователя, логин/пароль: tom@gmail.com/12345",bob@gmail.com/55555.
При нахождении на странице, сообщения об изменениях приходят в real-time,если в момент изменения конфигурации не быть на странице, то сообщения будут показаны при следующем входе
/api/ApiConfig post - создать новую запись 
/api/ApiConfig get - получить список конфигов, возмождно отфильтровать по названию или дате добавления(до или после указаной даты)
/api/ApiConfig put - отрелавтировать существубщую запись. поиск записи проиходит по навзани, версии и ключу. Меняется занчение


/login - реализована авторизация
/logout - разлогирование

/api/Subscribe post - для созданеия подписки неходимо указать корректные логин/пароль(tom@gmail.com/12345",bob@gmail.com/55555) и пит событий подписки вещественным числом           
{ 1, "All actions"},
{ 2, "Update only"},
/api/Subscribe delete - удалить подписку
